🍎 만능 다운로더 · 맥 단축키 설정

URL 복사 → 클릭(또는 단축키) 한 번 → 자동 다운로드. 5분이면 끝나요.

0단계 · 준비 (딱 한 번)

도구 설치 + 파일 권한

  1. 터미널을 열고(스팟라이트 ⌘Space → “터미널”) 아래를 붙여넣어 실행:
brew install yt-dlp ffmpeg gallery-dl

※ Homebrew가 없다면 먼저 brew.sh에서 설치.

  1. 받은 만능다운로더.command 파일을 ~/Downloads(다운로드) 폴더에 둔 뒤, 아래로 실행권한 부여 + 첫 실행 차단 해제:
chmod +x ~/Downloads/만능다운로더.command && xattr -d com.apple.quarantine ~/Downloads/만능다운로더.command 2>/dev/null; echo 완료

파일을 다른 폴더에 뒀다면 ~/Downloads/ 부분만 그 경로로 바꾸면 돼요.

방법 A · 제일 간단 (추천 · 무설정)

복사 → 파일 클릭

  1. 영상 페이지에서 URL을 복사(⌘C).
  2. 만능다운로더.command를 더블클릭 — 스크립트가 클립보드의 URL을 자동으로 집어 바로 받기 시작해요.
  3. 계속 다른 URL을 붙여넣어 받거나, q + Enter로 종료.

💡 파일을 Dock(독)에 끌어다 놓으면 매번 “복사 → 독 아이콘 클릭”으로 끝. 단축키가 굳이 필요 없어요.

첫 실행에서 “확인되지 않은 개발자” 경고가 뜨면: 파일 우클릭 → 열기 → 열기 한 번(0단계의 xattr을 했으면 안 떠요).

방법 B · 전역 단축키

단축어(Shortcuts) 앱으로 키보드 단축키 걸기

  1. 단축어 앱 실행(스팟라이트에서 “단축어”/“Shortcuts”).
  2. 왼쪽 위 + → 새 단축어 생성.
  3. 오른쪽 검색창에 “셸 스크립트 실행”(Run Shell Script) 검색 → 액션 추가.
  4. 그 액션 칸에 아래 한 줄을 붙여넣기:
open ~/Downloads/만능다운로더.command

이러면 단축키를 누를 때 터미널이 열리며 스크립트가 실행되고, 위처럼 클립보드 URL을 자동 인식해요. (진행 화면도 보여요.)

  1. 오른쪽 단축어 세부사항(ⓘ) 패널 → “키보드 단축키 추가” → 원하는 조합 지정(예: ⌃ ⌥ D).
  2. 끝! 이제 어디서든 URL 복사 → 그 단축키 = 자동 다운로드.

파일 경로가 다르면 ~/Downloads/만 바꾸면 됩니다.

문제 해결
  1. 화질이 계속 낮게 받아짐 → yt-dlp가 낡은 것. 아래로 최신화:
    brew upgrade yt-dlp
  2. “command not found” → 0단계의 brew install을 안 했거나 Homebrew 미설치.
  3. 받은 파일 위치 → 기본 저장 폴더는 ~/Downloads/yt-dlp. 구글드라이브가 켜져 있으면 …/My Drive/Shared에도 자동 복사돼요.
