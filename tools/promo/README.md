# tools/promo — 홍보 산출물 생성 파이프라인 (표·카드 이미지)

일러스트·PPT로 수작업하던 홍보용 표·안내 카드 이미지를 **말로 시켜서** 뽑는 도구.
운영자는 세션(Claude)에 내용만 던지면 된다 — 예: 『교육실 대관료 표처럼 ○○ 내용으로 뽑아줘』.

## 1. 세션(Claude) 작업 절차
- a) `templates/`에서 가장 가까운 템플릿 복사 → 내용 교체(원문 숫자·문구 그대로 — 임의 왜곡·창작 금지).
- b) 렌더: `node shot.js templates/price_table.html` (최초 1회 = 이 폴더에서 `npm i` · 한글 폰트 `apt-get install -y fonts-noto-cjk`).
- c) 출력(`out/*.png`)을 눈으로 실측(잘림·스크롤바·오타·한글 깨짐·JS 에러 0) 후 채팅에 파일 첨부.
- d) 기존 템플릿으로 안 되는 새 레이아웃 = 기존 패턴 최대 계승으로 시안 제작 → **운영자 승인 후에만** `templates/`에 승격(전례 누적). 결정 전 옵션 비교가 필요하면 라우터 §🎛 플레이그라운드 경로.

## 2. 규칙 (제1 절대명령 연동)
- a) 색·구조 토큰의 값 SSOT = `index.html` `:root` 2블록 — `shot.js`가 **렌더 시 자동 주입**한다. 템플릿에는 토큰 사본·raw 색 금지, `var()` 참조만. (그래서 템플릿을 단독으로 열면 무색 = 정상.)
- b) 팔레트 폐쇄형: `docs/디자인기틀.md` §0 컬러칩 밖 색 반입 금지. 기틀에 없는 값이 필요하면 멈추고 운영자에게 질문.
- c) 템플릿에 `vh` 단위 금지(렌더러가 콘텐츠 높이 자동 맞춤 = fullPage).
- d) 산출 규격: 폭 816px(카톡·인쇄 겸용) · @2x PNG. 폭 변경 = `--width=NNN`.

## 3. 파일
- `shot.js` — 헤드리스 렌더러(playwright-core · 크로미움 자동 탐색 · 템플릿 JS 에러 시 실패 처리).
- `templates/price_table.html` — 가격표(그룹 섹션 + 기본/추가 2컬럼). 운영자 승인 전례 1호.
- `out/` — 출력물(git 미추적 · 전달은 채팅 첨부로).

## 4. 후속 확장(운영자 결정 대기 — 원장 §우선사항 참조)

- C안 = Worker 경유 자연어 생성(Claude API — 비용·키 관리 필요).
