## 3. 묻고-편입 프로토콜 (기틀에 없는 게 필요할 때)

1. **멈춘다.** 임의로 raw hex·새 클래스·새 :root를 만들지 않는다.
2. **운영자에게 질문**: ① 왜 필요한지 ② 가장 가까운 기존 후보(토큰/컴포넌트)와 왜 부족한지 제시.
3. **승인되면 즉시 기틀 편입** (3점 세트):
   - `:root` 해당 블록에 토큰 추가 (raw hex는 토큰 정의부에만 존재)
   - 이 문서 인벤토리/토큰 표에 등재
   - `tools/check_design.py` baseline 갱신 + **사유 주석** (예: `# 1827→1829 = PR#XX 위험 배너 신설 +2(2607XX)`)
4. 거절되면 기존 후보로 구현.
