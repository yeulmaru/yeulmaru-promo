# dash_legacy — yeulmaru-dash 레포 폐기 전 보존 로직 (260611)

yeulmaru-dash(Streamlit)는 promo로 완전 이관 후 폐기됨. 미래 참조 가치가 있는 3개 모듈만 보존:

- `genre_classifier.py` — 운영대장 정산 입력 시 쓰던 **9단계 장르 분류 규칙** (연 단위 정산 행 추가 때 일관성 기준)
- `schema.py` — 운영 DB 컬럼 정의·장르 기본값·오타 매핑
- `data_cleaner.py` — 연도 정규화·오타 수정·취소 판별

⚠️ dash의 promo_sync(엑셀→운영_시트 미러링)는 **재실행 금지였음** — 정규화(공연ID) 이후 운영_* 시트가 단일 원본이며, dash 엑셀로 덮어쓰면 공연ID가 소실됨. (레포 폐기로 위험 해소)
운영 흐름: 일일입력=promo 폼 · 신규 공연=공연마스터 · 연간 정산=운영_세부운영관리대장정리에 직접 추가.
