⛔ 팔레트 폐쇄형 — 이 컬러칩이 전부다
아래 컬러칩 외의 색을 절대 들여오지 않는다. 다른 팔레트에서 색을 가져오거나, 기존 칩과 비슷한 색을 새로 만들어 칠하는 것 금지 — 유사색이 난립하면 브랜드가 무너진다. 이 폐쇄성이 브랜드의 핵심이다.
각 역할색은 딱 하나다. 새 색이 정말 필요하면 임의로 만들지 말고 운영자에게 확인한다. (같은 토큰의 alpha 변주·팔레트 안 동값 별칭 공존은 정상 — §3.5②)
