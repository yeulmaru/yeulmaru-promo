판매대행의뢰서 매크로 파일(xlsm) 개선

2026-07-10 · 운영자 업로드 파일 직접 수정(XML 외과수술 + VBA 재조립) · 개선판은 채팅으로 전달(개인정보 포함이라 레포 미커밋)

요약 — 뭐가 바뀌었나

① 분류별 행·열 구분 (시트 1.판매대행의뢰서)

입력 틀·값은 그대로 두고, 행을 대분류/소분류 2단 아웃라인 그룹으로 묶음(기본 = 전부 펼침). 좌측 여백의 [1][2][3] 버튼으로 한 번에 접고 펴고, 각 섹션 머리행 옆 +/−로 개별 토글. 참고용 목록 열(AK~AN)은 숨김 열 그룹으로 정리(AJ열 위 +로 펼침).

② 구분(공연/전시/예술교육) → 장소 연동 드롭다운

AH8(인쇄 영역 밖)에서 구분을 고르면 G8 공연 장소명 드롭다운 목록이 그 구분에 귀속된 장소로 바뀜. 장소 값 = 프로모 앱 실측 재사용. 목록 밖 장소(외부 대관 등)도 직접 입력 가능 — 경고만 뜨고 계속 진행.

③ 첨부 = 폴더 복사 + 순차 링크 (시트 3.이미지 사이즈표)

G열 더블클릭 → 파일 선택 → 엑셀 파일 옆 [첨부파일] 폴더가 자동 생성되고 순번_항목명_원본파일명으로 복사 + 셀에 클릭하면 열리는 링크. 같은 항목 재첨부 = 기존 첨부 교체. 기존 OLE 임베드 방식(엑셀 비대화·꺼내기 번거로움) 대체.

분류(그룹) 구조

[1] 상품 상세 정보 (4~37행) [2] 공연 정보 · 장르 · 주요 사항 · 기타 정보 · 검색어 · 등급/가격 · 할인 · 표기사항 [3] 각 소분류의 상세 입력행 [1] 기획사 기본사항 — 계약/정산 (38~43행) [1] 공연예술통합전산망 (44행) [1] 판매대행 수수료 (45~52행) [1] 관리자 페이지 (53~54행) [1] 개인정보 안내·유의사항 (55~62행)

목록 원본 = 시트1 숨김 열 AK13:AN14 + 정의된 이름 4개(구분목록·장소_공연·장소_전시·장소_예술교육) + INDIRECT 데이터 검증. 장소 추가는 해당 열에 값 넣고 정의된 이름 범위만 늘리면 됨.

기술 방식 — 왜 XML 수술인가

  • openpyxl 재저장은 이 파일의 양식 컨트롤(체크박스 51개)·서식을 유실할 수 있어 배제. zip 파트 5개만 교체하고 나머지는 바이트 그대로 보존: workbook.xml(정의이름) · sheet1.xml(Claude Log Turn 7) · sheet2.xml(그룹·드롭다운) · sharedStrings.xml · vbaProject.bin.
  • VBA는 MS-OVBA 압축기 + CFB(OLE) 라이터를 직접 구현해 Sheet4 모듈 소스만 교체. 성능 캐시 제거(MODULEOFFSET→0) + _VBA_PROJECT 버전 무효화로 열 때 전 모듈 소스 재컴파일 강제 — 낡은 p-code가 실행될 여지 차단.
  • 재적용 스크립트 = tools/scratch/upgrade_ticketlink_xlsm.py (원본 xlsm에 언제든 재실행 가능).

사용법 (운영자용 1분 요약)

  • 열 때 노란 보안 바에서 [콘텐츠 사용] — 매크로 활성화(더블클릭 첨부에 필요).
  • 시트1 좌측 상단 [1] = 대분류만 보기 · [2] = 소분류까지 · [3] = 전체 펼침.
  • AH8 구분 선택 → G8 장소 드롭다운이 그 구분의 장소로 바뀜.
  • 사이즈표 G열 더블클릭 = 첨부. 파일을 먼저 저장한 뒤 사용(저장 전이면 안내 후 중단). 링크 클릭 = 그 파일 열림. NHN 제출 시 [첨부파일] 폴더째 보내면 됨.

한계·유의

  • 실기 Excel 미검증: 구조는 표준 준수 + LibreOffice 검증까지 마쳤지만, 실제 Excel에서 열었을 때 복구 안내 등 이상이 보이면 즉시 알려줄 것 — 원본 보존돼 있어 재생성 빠름.
  • 첨부 매크로는 Windows Excel 기준(경로 구분자·FileDialog). Mac 사용 시 알려주면 분기 추가.
  • 첨부 후 파일을 다른 폴더로 옮기면 [첨부파일] 폴더도 같이 옮겨야 링크가 삶(상대 경로).
  • 앱의 의뢰서 생성 기능(reference/ticketlink_template.xlsx 기반)은 구판 템플릿을 씀 — 이번 개선판을 새 템플릿으로 승격하려면 셀 매핑 재배선이 별도 작업(운영자 지시 대기).
