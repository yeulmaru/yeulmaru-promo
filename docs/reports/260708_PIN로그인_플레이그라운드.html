#운영자 요구시안 반영
1입력 전 흰색 불투명도 85% (입력 느낌↑)칸 배경 rgba(255,255,255,.85) — 도트가 아니라 눌러 넣는 "칸"으로. 슬라이더로 조절
2입력하면 강조색 + 액티브 획채우면 var(--accent) 테두리+var(--accent-light) 배경, 밑줄 획이 슥 그려짐
3다 맞으면 테두리가 시계방향으로 동그랗게 감싸기각 칸 라운드 링이 시계방향 draw(순차) + 살짝 튀어오름
4입력 시 통통 튀는 번호숫자가 scale .5→1.18→1 바운스로 등장
5칸 아래 "PIN 번호를 입력해주세요" 잘 보이게강조색 굵은 안내문구 상시 노출(입력 완료되면 사라짐)
6CapsLock 잔재 삭제PIN은 숫자라 무의미 → 배선 시 .pin을 CapsLock 대상에서 제외(비밀번호 설정 모달엔 유지)
