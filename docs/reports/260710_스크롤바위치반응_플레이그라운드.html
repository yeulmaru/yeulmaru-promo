스크롤바 위치·이동 반응 — 플레이그라운드

판매현황(모달 #ana-body · 우측 레일 · 메인 창)에 뜨는 살몬 스크롤바를, 위치·이동에 반응하도록. 아래 패널을 직접 스크롤해 상태 전환을 확인 →. 신규 토큰·hex 0(2강조색=var(--c2) 계승 · 무채색=흑알파, 캘린더 스크롤바 rgba(0,0,0,.15)와 동일 계승).

동작 규칙 (해석 — 틀리면 알려줘)

① 최상단·최하단에 있을 때 → 2강조색(살몬), 지금처럼.
② 중간·정지 상태(기본) → 무채색 글래스, 투명도 높임(불투명도 ≈ 13%) — 유령처럼 은은.
③ 중간·이동중(스크롤/드래그) → 무채색, 조금 더 진하게(불투명도 ≈ 30%).
즉 위치가 색을 정하고(끝=살몬 / 중간=무채), 이동은 중간 무채의 진하기만 올린다. 끝단은 이동 여부와 무관하게 살몬.

⑤ 재현 한계 각주

· 글래스모피즘 근사: ::-webkit-scrollbar-thumb엔 backdrop-filter(실제 블러)가 안 먹는다 → "글래스"는 반투명 흑알파로 근사. 뒤 배경이 비쳐 유리 느낌만 냄(진짜 블러 아님).
· webkit 전용: Chrome·Edge·Safari에서 위 모양대로. Firefox는 scrollbar-color만 지원 → 썸/트랙 2색만 반영(화살표·상태전환 무).
· 이동중 판정: 휠·드래그·키보드 스크롤 모두 scroll 이벤트로 잡고, 멈춘 뒤 ~420ms에 정지로 복귀(디바운스).
· 끝단 우선: 이동 중 끝단 도달 시 = 살몬(끝단이 이동보다 우선). 규칙 ①대로.
