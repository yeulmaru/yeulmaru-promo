요약 — 이제 앱에서 뽑는 의뢰서에 개선분이 전부 실린다

바뀐 것 (사용자 관점)

  • 티켓 등록 신청 모달의 장소: 자유 입력 → 구분(공연/전시/예술교육) 연동 드롭다운 + 목록 밖 장소는 「직접 입력…」. PPT에서 「예울마루 대극장」처럼 읽혀도 정식 명칭·구분으로 자동 스냅.
  • 생성 파일 = .xlsm: 분류별 접이식 그룹([1][2][3]·±) · AH8 구분→G8 장소 드롭다운 · 사이즈표 더블클릭 첨부 매크로(폴더 복사+링크)까지 탑재된 채로 다운로드. 열 때 [콘텐츠 사용] 필요.
  • 공지문 장르 칸 버그 해소: 구판은 G9(장르 텍스트)를 안 채워 공지문 장르가 공란이었음 — 이제 기입.
