      재현 한계 각주

      · 원본은 React(motion/react) — 여기선 CSS keyframes + SMIL(모프) + JS 인터벌로 등가 이식(실배선도 이 방식, 라이브러리 불요). 타이밍·딜레이·진폭 = 원본 수치 그대로.

      · 색은 원본(currentColor) 구조 유지 — 지정은 기존 토큰·흰색만(칩 밖 색 없음). 흰색 = 현행 오버레이 글자·버튼 글자 계승.

      · 이징 cubic-bezier(0.77,0,0.175,1) = beui 원본 — 우리 기틀에 없는 신규 모션 커브(갱신 후보 ⚠). 토글로 기존 주력 (0.4,0,0.2,1) 계승판과 비교 가능.

      · Scramble 글리프는 랜덤(원본 동일 — 실행마다 다름) · Percent는 실제 진행률과 무관한 데모 루프(원본 동일).

      · reduced-motion 폴백(원본 = 투명도 펄스)은 여기선 미재현 — 실배선 시 포함 권장.

      · 갤러리 = 17종 상시 구동이라 실앱(로딩 중 1개)보다 CPU 사용 큼 · 오버레이 타자기 속도 65ms/자 = 실측.

      · 실앱 폰트 동일(시스템 스택) · 캘린더 스켈레톤 행높이만 132→118px 축소(프레임 470px 맞춤).
