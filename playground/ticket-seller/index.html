04 대관 공연 — 두 갈래 트랙

대관은 대관자의 선택지 2개로 표준화

우리 기획 공연은 아니지만 대관 공연도 발권까지 지원합니다. 판매 방식에 따라 트랙이 갈립니다.

Track A

정식 위탁판매

  1. 대관 계약 시 회당 좌석 50%+ 티켓링크 할당석 배정 (2027.01.01부터 강제)
  2. 대관자 ↔ 티켓링크 위탁판매계약 — 수수료·정산은 그 계약을 따름
  3. 네이버 채널은 티켓링크 독점 판매
  4. 판매·발권·정산은 기획공연과 동일 구조
Track B

등록 전용 (발권 지원)

  1. 대관자가 자체 판매 — 결제는 시스템 밖
  2. 공연 정보·좌석도·명단 → 예울마루 담당자 (공연 7일 전)
  3. 티켓링크가 등록 전용으로 상품 등록
  4. 현장 매표소에서 발권만 지원 — 발권수수료 미발생

05 질문별 상세 · 원인 · 해결책 · 회신문

문의 A(단체·초대권·보고서·사업ID) + 문의 B(결제·정산·수수료)

각 질문 = 결론 → 질문 원문 → 시스템 현황·원인 → 해결책 → 관련 화면 → 회신문(복붙용) 순. 화면 경로는 실서비스 seller.ticketlink.co.kr 기준.

시스템 현황·원인

셀러 시스템·공식 매뉴얼(31p·112p) 어디에도 단체 전용 모듈은 없습니다. 티켓링크의 단체 처리 모델이 전용 화면이 아니라 기존 기능의 조합이기 때문입니다. 실제로 예매 화면에는 예매수수료 부과와 함께 '단체' 옵션·금액 입력란이 이미 있습니다(셀러가이드 p.26). 달샤베트 의뢰서에도 "10/1·10/2 11시 = 단체 전용 회차", "[4] 단체할인: 전화 예매만 가능(10인+ 47%, 50인+ 50%)"로 이 모델이 반영돼 있습니다.

해결책 (운영 체계)
  • ① 단체 전용 회차 — 특정 회차를 개인 판매에서 숨기고 단체 배정.
  • ② 전화(대리) 예매 — 셀러 예매 화면에서 무통장(가상계좌) 대리 예매, 단체할인 적용, ARS 수기결제 지원. 입금 확인은 내역조회 '예매대기(미입금) 포함' 조건.
  • ③ 일괄 발권 — 방문 당일 전체예매발권으로 단체분 한 번에 출력.
  • B2B 코드 — 기관별 전용 URL 발급 시 학교·관공서 셀프 결제 채널로 활용(여수시청 코드 기운영).

회신문 (복붙용)

티켓링크 시스템에 단체 전용 예약 프로그램(별도 모듈)은 제공되지 않습니다. 다만 아래 3가지 조합으로 단체 예매를 운영할 수 있으며, 당관은 이를 표준 프로세스로 정립하겠습니다. ① 단체 전용 회차 운영 ② 담당자가 셀러 시스템 예매 화면에서 단체 건을 무통장으로 등록(단체할인 10인 47%·50인 50% 적용) ③ 방문 당일 전체예매발권으로 일괄 출력. 학교·관공서가 직접 결제를 원할 경우 기관별 B2B 전용 URL(할인코드) 발급으로 셀프 예매 채널을 열 수 있습니다.

세금계산서 · 원인

공연 입장권은 부가세 면세 용역이라 세금계산서 발행 대상이 아닙니다. 관공서·학교의 지출증빙 요구는 ① 지출증빙용 현금영수증(사업자번호) 또는 ② 카드매출전표로 충족됩니다. 증빙 권한이 3주체(티켓링크 시스템 / PG(페이코) / 예울마루 재무)로 분산돼 있어 문의마다 개별 대응이 필요했던 것이 근본 원인 — 아래 매트릭스를 공식 안내문으로 배포하면 해소됩니다.

시스템 증빙은 현장 판매 건 기준(셀러가이드 p.87). 온라인 예매 건은 고객이 티켓링크 마이페이지에서 직접 출력.

A-2 · 모바일 초대권
대관 공연 초대권(교환권)의 모바일 발행
가능 — 모바일 티켓 발송 인프라 실재. 비용 150원/장 + 디지털티켓 건별 견적
질문 원문대관 공연 초대권(교환권)을 모바일 티켓 형태로 발행 가능한지? 발행 방식 및 사용 절차(비용)는?
발행 방식
  • 상품 등록 시 초대권(0원) 권종 + 모바일 티켓 사용 명기 (이미지 600×368 규격 제출). 0원 권종 추가는 티켓링크 운영팀 요청.
  • 수령자 명단을 엑셀로 업로드 → 시스템의 모바일 티켓 발송 기능이 알림톡(카카오)·이메일로 즉시/예약 발송(셀러가이드 p.84~85).
  • 관람 당일 모바일 바코드를 입장처리 스캔으로 바로 입장(실물 교환 불요).
비용 · 원인

초대권·교환권·0원권 발권은 계약상 장당 150원(VAT 포함)(② 제3조 6항). 온라인 판매 티켓 발권은 무료(② 제4조 1항). 모바일(디지털) 티켓 제작·발급은 티켓링크 재위탁사 ㈜부스터랩 수행이라 수량·형태별 건별 견적이 필요 — 확정 수량으로 사전 요청 시 견적 안내. 단, 등록 전용 상품은 발권료 미발생.

실물 교환권 병행도 가능(당일권/기간권). 교환권은 발권 취소 불가에 유의(셀러가이드 p.18). 발행·사용 현황은 유무료 판매집계·교환권 발행내역으로 리포팅.

회신문 (복붙용)

대관 공연 초대권(교환권)은 모바일 티켓 형태로 발행 가능합니다. 발행 방식: 상품 등록 시 초대권(0원) 권종과 모바일 티켓 사용을 명기 → 수령자 명단을 엑셀로 업로드 → 시스템이 알림톡/이메일로 즉시·예약 발송 → 관람 당일 모바일 바코드를 현장 스캔으로 입장. 비용: 초대권·교환권·0원권 발권 시 계약상 장당 150원(VAT 포함)이 발생하며, 모바일(디지털) 티켓 제작·발급은 제휴사를 통해 진행되어 수량·형태에 따라 건별 견적이 필요합니다. 실물 교환권 병행도 가능하며 발행·사용 현황은 보고서로 제공합니다.

티켓링크 확인 요청 — 모바일 초대권 발급비(디지털 티켓 단가), 명단 업로드 포맷, 알림톡 발송 비용 부담 주체.

A-4 · 사업별 ID
공연·교육·전시·대관 별도 ID(관리코드) 운영
가능 — 관리코드 규칙(즉시) + 파트너 계정 분리(선택). 매출·통계·정산 분리 관리
질문 원문사업 유형별(공연/교육/전시/대관)로 별도 ID(관리코드)를 부여해 운영할 수 있는지? 사업별 매출·통계·정산 데이터가 분리 관리되는지?
시스템 현황·원인

시스템은 이미 멀티 파트너 전환 구조(판매처 셀렉트 + changePartner)와 창구(1~10, 90번대) 구분을 지원하고, 로그인 시 "채널검색→창구 선택" 절차가 그 증거. 상품에는 관리 카테고리·상품코드·장르(공연/전시/스포츠) 필드가 이미 붙어 보고서 곳곳에 컬럼으로 나옵니다(셀러가이드 p.51·p.53·p.65). 다만 현재는 단일 계정(reservePartnerNo=27247)으로만 운영 중이라 규칙이 없을 뿐 — 상품 등록·관리코드 부여는 티켓링크 내부(nbs 상품등록)에서 의뢰서 기반 수행.

해결책 2단계
  • ① 관리코드 규칙 (즉시) — YWM-PF(공연)/EX(전시)/ED(교육)/RN(대관)-연도-일련. 의뢰서 작성 때 강제. 보고서 상품별 통합 집계 + 유무료 판매집계의 상품코드 컬럼으로 사업별 매출·통계 분리 집계.
  • 정산은 행사(상품) 단위로 완결되므로 코드만 정해두면 정산 데이터도 자연 분리. 전시·교육은 D-1 전용이라 채널 축에서도 이미 분리.
  • ② 파트너 계정 분리 (선택) — 완전 분리(정산계좌·보고서·창구)가 필요한 사업만 티켓링크에 별도 파트너 ID 발급 요청(추가 계정은 담당자 요청으로 가능, 매뉴얼 p.4).

회신문 (복붙용)

사업 유형별 구분 운영이 가능하며 2가지 방식을 병행 적용하겠습니다. ① 관리코드 체계(즉시): 상품 등록 시 관리코드를 「YWM-PF(공연)/EX(전시)/ED(교육)/RN(대관)-연도-일련」 규칙으로 부여합니다. 시스템 보고서(상품별 통합 집계, 유무료 판매집계의 상품코드 컬럼)에서 사업별 매출·통계가 분리 집계되고, 정산이 행사(상품) 단위이므로 정산 데이터도 코드 기준 분리됩니다. ② 파트너 계정 분리(선택): 정산 계좌·보고서·창구 단위 완전 분리가 필요한 사업은 티켓링크에 별도 파트너 ID 발급을 요청할 수 있습니다. 우선 ①로 운영하고 필요 사업만 ②를 적용하는 것을 권장합니다.

상품 등록 시 1인 N매 제한을 설정하며 온라인 채널에만 적용됩니다(현장 발권은 이 제한과 무관). 보안문자(클린예매) 옵션도 함께 설정 가능. 셀러 화면엔 직접 설정 기능이 없고 의뢰서→상품 등록 단계에서 티켓링크가 설정합니다.

② 방문일자 지정 — 불가 · 원인

전시 상품은 유효기간 내 1회 방문하는 기간권 모델로 등록되어 방문일자 필드 자체가 없습니다(비지정발권도 기간권 상품은 일자 축에서 비노출, 셀러가이드 p.11). 실계정 전시 상품(productId 64250)도 실제 기간 관람 모델로 확인. 따라서 '지정일 이후 취소 불가'도 걸 대상이 없어 불가. 취소는 계약 기준(관람 유효기간 종료 전일 17시)을 따릅니다.

대안 2가지
  • 회차형 등록 — 전시를 일자별 회차 상품(회차당 정원=일일 수용량)으로 등록하면 일자 지정·해당일 전일 17시 취소 구조가 됩니다(티켓링크 협의).
  • 하이브리드 — 티켓은 기간권으로 팔되 예울마루 홈페이지에서 방문 예약(날짜 슬롯)만 자체 접수. 티켓 취소 정책과 분리돼 리스크 없음.

매수제한 — 가능합니다. 상품 등록 시 "1인 N매" 제한을 설정하며(현장 발권과는 별개로 온라인 채널에만 적용), 보안문자(클린예매) 옵션도 함께 설정할 수 있습니다. 희망 매수를 알려주시면 의뢰서에 반영합니다.

회신문 (복붙용)

수수료는 성격이 다른 2종이므로 구분해 안내드립니다. ① 판매(결제)수수료(예울마루 부담): 홈페이지 전용으로 열어도 결제수단별 수수료가 발생합니다 — 무통장 0% / 카드 3% / 네이버페이 6%(부가세 별도, 티켓 실판매가 기준). 전시·교육은 홈페이지 전용이므로 이 요율만 적용됩니다. ② 예매수수료 2,000원/매(고객 부담): 예울마루 홈페이지 예매와 현장 발권은 면제이며, 티켓링크·네이버 등 외부 채널 예매 시에만 부과됩니다. ③ 외부 링크로 티켓링크 예매 페이지가 열리는 경우 이를 홈페이지로 강제 이동시키는 것은 현재 불가하나 조율 여지가 있어 협의 중입니다.
